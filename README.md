# PUBG Auto Box Opener

一个用于 PUBG 游戏的自动开宝箱工具，支持模板匹配和固定坐标两种模式，具备智能等待、多尺度匹配、统计分析等功能。

## 功能特点

- **双模式支持**：模板匹配模式（推荐）和固定坐标模式
- **智能等待**：轮询检测关闭按钮，而非固定等待时间
- **多尺度匹配**：适应不同游戏分辨率
- **开箱统计**：实时显示开箱数量、失败次数、用时和速率
- **日志记录**：运行日志保存到文件，方便排查问题
- **配置持久化**：参数自动保存到配置文件
- **暂停/恢复**：支持快捷键暂停和恢复
- **内置辅助工具**：模板截取、坐标获取

## 安装要求

- Python 3.7+
- 依赖库：
  - pyautogui - 屏幕操作和鼠标控制
  - opencv-python - 图像处理和模板匹配
  - pillow - 图像保存
  - keyboard - 键盘监听

## 安装依赖

```bash
pip install pyautogui opencv-python pillow keyboard
```

### 模式说明

#### 1. 模板匹配模式（推荐）

需要在 `templates/` 目录中添加模板图片：

- `open_button.png` - 右键宝箱后出现的"开启"按钮截图
- `close_button.png` - 开箱完成后出现的"关闭"按钮截图
- `chest.png` - 宝箱图标截图（可选，用于自动右键点击）

#### 2. 固定坐标模式

手动获取按钮坐标，适合分辨率固定的场景。

### 快捷键

| 按键  | 功能    |
| --- | ----- |
| F12 | 停止运行  |
| F11 | 暂停/恢复 |

## 配置说明

配置文件 `box_opener_config.json` 包含以下参数：

| 参数                        | 类型    | 默认值         | 说明               |
| ------------------------- | ----- | ----------- | ---------------- |
| confidence\_threshold     | float | 0.7         | 模板匹配置信度阈值（0-1）   |
| scan\_interval            | float | 0.5         | 扫描间隔（秒）          |
| click\_delay              | float | 0.2         | 点击延迟（秒）          |
| animation\_timeout        | float | 8.0         | 开箱动画超时（秒）        |
| animation\_poll\_interval | float | 0.3         | 轮询检测间隔（秒）        |
| retry\_count              | int   | 3           | 重试次数             |
| multi\_scale              | bool  | true        | 是否启用多尺度匹配        |
| scale\_range              | list  | \[0.8, 1.2] | 缩放范围             |
| scale\_steps              | int   | 5           | 缩放步数             |
| log\_to\_file             | bool  | true        | 是否记录日志到文件        |
| capture\_region           | list  | null        | 截图区域（x, y, w, h） |
| open\_coords              | list  | null        | 开启按钮坐标           |
| close\_coords             | list  | null        | 关闭按钮坐标           |
| chest\_coords             | list  | null        | 宝箱坐标             |

## 快速开始指南

### 首次使用（模板匹配模式）

1. 运行脚本，选择 **选项 3** 截取模板图片
2. 按照提示用鼠标框选"开启按钮"和"关闭按钮"区域
3. 选择 **选项 1** 开始自动开箱
4. 按 F12 停止运行

### 首次使用（固定坐标模式）

1. 运行脚本，选择 **选项 2**
2. 按照提示依次获取开启按钮、关闭按钮坐标
3. 选择是否获取宝箱坐标（用于自动右键）
4. 开始自动开箱

## 注意事项

1. **游戏设置**：确保游戏窗口在前台显示
2. **分辨率**：模板图片需要与游戏分辨率匹配
3. **安全设置**：`pyautogui.FAILSAFE = True`，将鼠标移到屏幕左上角可紧急停止
4. **权限**：需要管理员权限才能正常监听键盘事件
5. **合规性**：请遵守游戏使用条款，合理使用本工具
